## Website using:

- Heroku (Free host)

## How it work:

```
        using                                  call
User <----------> Angular <----------> API <----------> cloud mongoDB
                            get/post
```

<hr>

## Installation
1. Clone the project
2. Install package
    ```
    npm install
    ```
3. Run localhost
    ```
    ng serve --open
    ```
