## Website using:
- [Angular](https://angular.io/) (Web server)
- [Flask on python](https://www.fullstackpython.com/) (API)

- [Heroku](https://dashboard.heroku.com/) (Free host)

## Installation
1. Clone the project
2. Install package
    ```
    npm install
    ```
3. Run localhost
    ```
    ng serve --open
    ```
